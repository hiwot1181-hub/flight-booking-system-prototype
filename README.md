#Flight Booking System - Academic Web Application Prototype
## Overview
This project is a full-stack web application prototype developed as a part of an Internet Programming course. It simmulates a basic flight booking system including user interface design, booking functionality, and database integration.
## Features
- Flight search interface
- Booking form simulation
- Backend processing using PHP
- Database integration for booking data storage
## Technologies Used
- HTML
- CSS JavaScript
- PHP
- MYSQL
## Purpose
This academic project was developed to demonstrate understanding of full-stack web development fundamentales including frontend design, backend logic, and database connectivity.
## Status
Prototype version - created for educational purposes.
